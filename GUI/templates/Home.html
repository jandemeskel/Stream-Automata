Formalisation of W-Automata

Input stipulation


(1) Enter all states, seperating each with a full stop and labelling them with Alphabetic only or an indexed tuple

(2) Enter initial states, only from the set of states entered above

(3) Enter the collection of accepting sets as a nested array, Where each array is a seperate set, and each state within a set is seperated by the boolean operator or, '|'.

(4) Enter all characters within the computation alphabet, soley represented by numbers.

(5) Enter the transitions between states using the format [start state. [input characters]. [end states]], leaving a hypthen between each transition array.

Accepting: [[A|B]]
A collection containing a single sets of accepting conditions, A and B.

Transitions [A. [0|1]. [A|B]] - [B. [0|1]. [A|B]]
Transitions from states A & B.
State transitions from state A to A and B via computing 0 and 1 respectively.
State transitions from state B to A and B via computing 0 and 1 respectively.
